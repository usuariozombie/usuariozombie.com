Más información

Portfolio personal para presentar próximos proyecto o ideas para lanzar al público, entre ellos cualquier aplicación que se encuentre en mi GitHub personal o en su defecto proyectos aún no lanzados. De los últimos serán presentadas sus funcionalidades y propósito en esta misma página para su seguimiento.

Proceso

Todas las aplicaciones se desarrollan hasta un punto en el que son totalmente funcionales para el público, aun así se puede acceder a ellas como beta tester para probarlas y dar feedback al desarrollador, facilitando así la corrección de bugs.

Idea

Las ideas para las aplicaciones se basan en la necesidad de los usuarios, por lo que la mayoría de ellas son de uso personal, aunque también se pueden infinidad de ideas entre ellas bots para empresas, páginas web, etc...

Objetivo

La meta de este proyecto es poder crear una marca personal que pueda ser reconocida por el público, para así poder ofrecer servicios de desarrollo de aplicaciones, creando también codigo abierto disponible para todo el mundo.

Misión

La misión es completar la funcionalidad de las aplicaciones y otros desarrollos para que puedan ser utilizadas sin fallos, además de crear una comunidad de usuarios que puedan probar las aplicaciones y aportar ideas y código para mejorarlas.

Conóceme

Llevo adquiriendo conocimientos de programación desde los 16 años, me he dedicado a ponerlos en práctica en varios proyectos desde entonces, se pueden destacar varias etapas de desarrollo dentro de mi vida. Podría decir que mi inicio fue con LUA, un lenguaje de programación que se utiliza en el juego Garry's Mod, en el cual sobre todo me dediqué a editar plugins ya que mi nivel no era muy avanzado. Más tarde aprendí Python y me dediqué a crear bots para Discord que están disponibles en mi GitHub, también creé aplicaciones de escritorio con Python muy básicas. Más tarde aprendí HTML y CSS y desarrollé páginas web. A día de hoy estoy aprendiendo JavaScript, he aprendido Java y estoy desarrollándome un poco más en LUA además de querer aprender C# y C++.
